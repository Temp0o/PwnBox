PWNBOX

ESP32-C5 Firmware Flasher
Requires Chrome or Edge on desktop. Won't work on Firefox/Safari/mobile.
How to flash:
1. Plug your ESP32-C5 into USB
2. Click CONNECT above
3. Pick the serial port in the popup
4. Click INSTALL and wait ~1 minute
5. Done — PwnBox boots on the device
If connection fails: hold the BOOT button on the dev board while plugging in USB, then release. This forces download mode. Make sure no other program (Arduino IDE serial monitor) has the port open.
